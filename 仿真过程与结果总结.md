# 多栅格滚球式 TENG 仿真过程与结果总结

## 1. 仿真目的

本次仿真围绕“双电极、四电极、六电极多栅格滚球式摩擦纳米发电机（TENG）”展开，目的是比较在相同 PTFE 滚球材料、相同滚球数量、相同运动输入条件下，不同电极结构对转移电荷、电流、电压、功率和能量输出的影响。

专利中的核心机理为：PTFE 滚球表面摩擦电荷密度保持基本一致，多栅格交替电极并不是提高滚球自身带电量，而是通过增加滚球运动过程中经历的有效电荷转移过程，使总转移电荷和能量输出提高。

## 2. 已采用的实验与结构参数

### 2.1 阵列结构

实验对象为双电极滚球式 TENG 阵列：

```text
阵列规模：16 × 9 个双电极单元并联
并联单元数：144
PCB 厚度：1 mm
单个双电极单元：A 电极 + 绝缘间隔 + B 电极
单电极宽度：16 mm
电极间距：5 mm
PTFE 球：每个单元 4 个，直径 4 mm
```

### 2.2 运动参数

```text
运动距离：100 mm
加速度：5 m/s²
减速度：5 m/s²
起点停留时间：30 ms
终点停留时间：30 ms
仿真周期：完整往复周期
```

完整周期定义为：

```text
起点停留
→ 正向加速/减速运动 100 mm
→ 终点停留
→ 反向加速/减速返回 100 mm
→ 回到起点停留
```

### 2.3 实测校准数据

```text
双电极转移电荷：380 nC
四电极转移电荷：650 nC
短路电流最高峰值：不同结构下基本接近
双电极最佳负载附近：约 1 GΩ
```

四电极转移电荷高于双电极的原因解释为：四电极结构在单次运动过程中发生多次有效电荷转移，因此总转移电荷增加；但由于实际运动状态、边缘电场、寄生电容和不完全复位等影响，提升并非理想的 2 倍。

## 3. COMSOL 模型建立过程

### 3.1 COMSOL 后端

已启动并使用 COMSOL 6.3 后端进行静电场求解。COMSOL MCP 曾用于检查服务状态和模型管理，但后续批量求解主要通过 COMSOL 的 Python/Java API 连接 `localhost:2036` 执行。

### 3.2 几何模型

已生成双电极、四电极、六电极的 2D COMSOL 几何模型：

```text

comsol_models/rolling_teng_N4_geometry_study.mph
comsol_models/rolling_teng_N6_geometry_study.mph
```

模型包含：

```text

```

### 3.3 最新 COMSOL 场仿真流程

最新的 COMSOL 场仿真脚本为：

```text
scripts/comsol_position_sweep_teng.py
```

该脚本执行如下流程：

1. 对双电极、四电极、六电极分别建立 2D 静电模型。
2. 在 PTFE 球边界施加固定表面电荷密度：

```text
σ = -50 μC/m²
```

3. 将所有电极边界设置为接地边界，用于计算瞬时感应电荷分布。
4. 对滚球位置进行参数扫描。
5. 对 A 端电极边界法向电位移进行积分，得到感应电荷：

```text
Q_A(x) = ∫ D_n dS
```

6. 用双电极实测转移电荷 `380 nC` 对 COMSOL 原始电荷结果做全局幅值标定。
7. 将位置函数 `Q(x)` 映射到完整往复运动轨迹 `x(t)`。
8. 计算：

```text
I(t) = dQ(t)/dt
V(t) = I(t) × R
P(t) = V(t)² / R
E(t) = ∫P(t)dt
```

其中当前负载为：

```text
R = 1 GΩ
```

### 4.1 COMSOL 位置扫描结果

在使用双电极 `380 nC` 做全局标定后，COMSOL 场模型得到：

| 结构 | COMSOL 标定后转移电荷 |
|---|---:|
| 双电极 | 380 nC |
| 四电极 | 约 387 nC |
| 六电极 | 约 388 nC |

该结果表明，目前的简化 COMSOL 静电模型主要反映“固定带电滚球在接地电极上方移动时的瞬时感应电荷变化”，还没有充分捕捉实验中四电极结构因多次电荷转移而达到 `650 nC` 的现象。

### 4.2 1 GΩ 负载下的 COMSOL 后处理输出

当前 COMSOL 场仿真后处理结果显示：

```text
双电极：Q = 380 nC
四电极：Q ≈ 387 nC
六电极：Q ≈ 388 nC
```

对应的电流、电压、功率和能量曲线已输出为 SVG 图。由于模型尚未引入真实端子历史、电荷转移复位过程和外电路耦合机制，四电极、六电极的转移电荷提升在该 COMSOL 场模型中尚不明显。

## 5. 实验校准等效模型结果

除 COMSOL 场仿真外，还建立了实验校准等效模型，用于反映实验观察到的多次电荷转移现象。

脚本为：

```text
scripts/simulate_multigrid_teng_calibrated.py
```

该模型采用：

```text
双电极：1 次主要电荷转移，Q = 380 nC
四电极：2 次非等幅电荷转移，总 Q = 650 nC
六电极：3 次非等幅电荷转移，按四电极衰减趋势预测
```

其中四电极第二个脉冲相对第一个脉冲的权重为：

```text
r = 650 / 380 - 1 ≈ 0.71
```

六电极脉冲权重预测为：

```text
[1.00, 0.71, 0.50]
```

对应预测：

```text
Q2 = 380 nC
Q4 = 650 nC
Q6 ≈ 842 nC
```

该模型更符合实验事实：不同结构的最高短路电流峰值接近，但多电极结构产生更多非等幅电流脉冲，因此总转移电荷和输出能量增加。

主要输出目录为：

```text
results_multigrid_calibrated/
```

重要输出文件：

```text
short_circuit_current_comparison_full_cycle.svg
short_circuit_current_N4_full_cycle.svg
current_comparison_1G.svg
```

## 6. 当前结论

1. 实验结果表明，多栅格结构提高总转移电荷的主要原因是增加了有效电荷转移次数，而不是提高 PTFE 球表面电荷密度。

2. 四电极结构相较双电极结构具有更高的总转移电荷：

```text
Q4 / Q2 = 650 / 380 ≈ 1.71
```

3. 四电极未达到理想 2 倍，是由于实际运动状态、边缘电场、寄生电容、电荷泄漏以及多次转移过程不完全等效等因素影响。

4. 当前 COMSOL 静电场模型已经建立了固定电荷密度下的场求解流程，但由于仅为瞬时静电感应模型，尚未包含端子历史和多次电荷转移机制，因此无法直接再现实验中的 `Q4 = 650 nC`。

5. 实验校准等效模型能更好体现专利中的工作机理，即：

```text
多栅格交替电极 → 多次非等幅电荷转移 → 总转移电荷增加 → 平均功率和周期能量提高
```

## 7. 对专利撰写的建议表述

建议在专利中将仿真与实验关系表述为：

> 基于 COMSOL 建立双电极、四电极及六电极结构的静电场模型，并结合实测转移电荷和负载输出数据建立等效后处理模型，对不同电极结构下的电荷、电流、电压、功率和能量进行对比分析。仿真和实验均表明，在 PTFE 滚球材料、滚球数量及运动激励条件相同的情况下，多栅格交替电极结构能够增加单个机械周期内的有效电荷转移次数，从而提高总转移电荷和输出能量。

不建议直接写为：

> COMSOL 直接得到四电极转移电荷 650 nC。

更准确的表述应为：

> COMSOL 静电场模型用于分析不同电极结构下的感应电场和电荷分布；实测转移电荷用于校准等效电荷转移模型，从而获得更符合实际工作过程的输出波形和能量预测。

## 8. 后续模型改进方向

为了让 COMSOL 结果更直接支持专利中的定量结论，建议后续升级模型：

1. 使用 Terminal 或 Floating Potential 边界，而不是全部电极接地。
2. 在 COMSOL 中加入外接负载或电路接口。
3. 对 A/B 电极端子电荷进行时序追踪，而不仅是瞬时静电积分。
4. 引入滚球跨越多个电极边界时的电荷转移历史状态。
5. 对四电极和六电极分别提取每个 A/B 边界的局部感应电荷变化，再进行完整周期叠加。
6. 将 COMSOL 场仿真与实测双电极、四电极数据联合校准，预测六电极结构。

当前最适合专利使用的结论是定性和半定量趋势：

```text
在相同 PTFE 电荷密度和相同机械输入下，多栅格交替电极通过增加有效电荷转移次数，提高转移电荷、平均功率和输出能量。
```

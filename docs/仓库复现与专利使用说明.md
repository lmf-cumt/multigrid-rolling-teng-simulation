上述图分别为四电极滚球式 TENG 在完整往复周期下的实测对齐源电流波形、转移电荷-时间曲线和 1 GΩ 派生负载响应。专利或实测对齐时优先使用源电流图，使用独立冻结脚本生成：

## 2. 当前冻结模型

冻结模型用于避免复现时受到主仿真脚本继续迭代的影响。参数如下：

| 参数 | 数值 |
|---|---:|
| 电极结构 | A-B-A-B |
| 电荷源模型 | 实测周期模板校准 |
| 电流图口径 | 源电流/电荷计电流 |
| PTFE 电荷密度 | 固定 |
| 四电极实测转移电荷 | 650 nC |
| 运动距离 | 100 mm |
| 加速度/减速度 | 4 m/s² |
| 完整周期 | 0.832456 s |
| 终点停留 | 100 ms |
| 起点停留 | 100 ms |
| 派生负载响应 | R = 1 GΩ，Ceq = 112.2 pF |

## 3. 为什么固定 PTFE 电荷密度

专利中的技术效果应归因于电极结构优化，而不是滚球材料本身带电更多。推荐表述为：

> 在相同 PTFE 滚球材料、滚球数量和运动激励条件下，滚球表面摩擦电荷密度基本保持一致。四电极交替结构通过增加 A/B 电极组之间的有效交替边界，改变静电感应分布并提高电荷转移利用率，从而获得更高转移电荷和输出电流。

## 5. 与 COMSOL 的关系

COMSOL 模型用于保存几何、电极排布、静电物理场和网格入口。当前电流图是基于实验转移电荷校准的等效电荷转移模型生成，不建议在专利中表述为“COMSOL 直接瞬态求解得到电流波形”。

推荐表述为：

> 建立多栅格交替电极结构的准静态电场模型，并结合实测转移电荷对等效电荷转移模型进行标定，获得完整运动周期下的输出电流变化规律。
